# Roadmap

## 0.1 MVP

- Self-hosted relay with Docker Compose.
- Desktop host with demo adapter.
- Android app can select desktop, select agent, send command, and receive realtime output.
- Basic audit log and device revocation.

## 0.2 Agent Integrations

- Hermes adapter.
- Agent Zero adapter.
- OpenClaw adapter.
- Adapter SDK documentation and examples.

## 0.3 Security Hardening

- QR/coded pairing.
- End-to-end encrypted command payloads.
- Per-adapter permission profiles.
- Replay protection and idempotency keys.

## 0.4 Distribution

- Signed Android APK release workflow.
- F-Droid metadata.
- Desktop host packages for Windows, Linux, and macOS.
- Published backend container images.
